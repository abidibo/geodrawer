geodrawer

The perfect library to use in the admin area of a modern web application in order to provide geolocalization info to all kind of entities.
Highly customizable and written focusing on the ease of use, can be easily integrated inside every project.

No css or static assets import, only a single js file (around 50 kb).

For info and installation guide please read the reference.
